## 기능구현 목록
### 입력 기능
- 코치 이름들 입력받기
  - 각각 최소 2글자, 최대 4글자
  - 코치들 최소 2명, 최대 5명
  - `,`로 구분
  - 중복 있으면 안됨
- 각 코치가 못 먹는 메뉴 입력받기
  - 최소 0개, 최대 2개
  - `,`로 구분
  - 중복 있으면 안됨

### 출력 기능
- 서비스 시작 문구 출력
- 메뉴 추천 문구 출력
- 서비스 종료 문구 출력

### 메뉴 추천 기능
- 한 주에 같은 카테고리는 최대 2회까지만 가능
- 각 코치에게 한 주에 중복되지 않는 메뉴를 추천

### ETC
- 에러 발생 시, 적절한 문구 출력 후 그 부분부터 입력
